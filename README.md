## 1.tailwindcss知识点复习
1. inset-0
inset是top,right.bottom.left的缩写。
`inset-0` 相当于 `top:0;right:0;bottom:0;left:0;`
2. flex-col
flex-col是flex-direction:column的缩写。colom是垂直方向。
3. flex-row
flex-row是flex-direction:row的缩写。row是水平方向，css默认就是水平方向。
4. animate-blink
animate-blink是用于实现闪烁效果的动画类。
